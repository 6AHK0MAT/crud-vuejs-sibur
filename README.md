# crud-vuejs-sibur

> Crud Vue.js e json-server

## Setup

```bash
# global install
npm install -g json-server

git clone https://github.com/6AHK0MAT/crud-vuejs-sibur
cd crud-vuejs-sibur

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```

Server wiil be on `http://localhost:3000`.
